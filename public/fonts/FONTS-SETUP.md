# Font files for this project

Place your font files here so they load from `styles/globals.css`.

## Sage Grotesk (headings)

In `public/fonts/sage_grotesk/` add one of:

- `SageGrotesk.woff2` (preferred)
- `SageGrotesk.woff`
- `SageGrotesk.ttf`

## Martian Mono (body and optional widths)

In `public/fonts/martian_mono/` add at least:

- **Body (required):** `MartianMono-StdRg.woff2` or `MartianMono-StdRg.ttf` (Standard width, Regular weight)

Optional (for badges, captions, emphasis):

- `MartianMono-CnRg` — Condensed
- `MartianMono-NrRg` — Narrow  
- `MartianMono-WdRg` — Wide

Use the same filenames with `.woff2` or `.ttf`. Download from [evilmartians/mono releases](https://github.com/evilmartians/mono/releases) and copy the needed files from the `ttf/` (or `woff2/` if available) folder.
